# NYC-Airbnb-Data-Analysis-
Exploratory Data Analysis (EDA) of NYC Airbnb listings using Python. Includes data cleaning, handling missing values, and visualizing market trends using Pandas, Seaborn, and Matplotlib.


Project Overview
This project performs a comprehensive Exploratory Data Analysis (EDA) on the New York City Airbnb dataset (100k+ listings). The goal is to uncover insights into pricing trends, host behavior, and neighborhood popularity while cleaning and preparing raw data for analysis.

Key Features
Data Cleaning: Handled over 50,000 missing values and addressed data type inconsistencies.

Outlier Detection: Identified and treated anomalies in pricing and availability.

Feature Engineering: Converted date strings to datetime objects and extracted temporal features.

Visualizations: Created insightful plots to show the growth of reviews over time and the distribution of listings across boroughs.

Tech Stack
Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn

Environment: Jupyter Notebook

Dataset Insights
The analysis covers:

Price distribution across Manhattan, Brooklyn, Queens, Bronx, and Staten Island.

Relationship between service fees and total listing price.

Impact of host verification status on listing reviews.
